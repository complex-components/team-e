# ImageBoard
A complex component for an image-based comment box.

## Usage
```javascript
import Imageboard from './imageboard'

ReactDOM.render(
  <Imageboard />,
  document.getElementById('imageboard')
)
```

## Styling
`ImageBoard` accepts a `style` property, so you can customize the look and feel.
